一些高级类型

  • Readonly: 只读类型
  • partial: 将类型定义的所有属性都修改为可选
  • Required: 将类型定义的所有属性都修改为必填
  • Pick: 将类型定义的部分属性作为一个新类型
  • Record: 以 typeof 格式快速创建一个类型，此类型包含一组指定的属性且都是必填。
  • Exclude: 排除
  • Omit: 省略
